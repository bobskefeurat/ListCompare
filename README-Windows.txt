ListCompare for Windows

1. Extract the entire zip file before running the program.
2. Keep ListCompare.exe together with the _internal folder.
3. Start the app by double-clicking ListCompare.exe.
4. The app opens in your web browser. If it does not open automatically, go to:
   http://127.0.0.1:8501
5. When you close the last browser tab or window for the app, ListCompare shuts down automatically after a short delay.

Notes
- User data is stored in %LOCALAPPDATA%\ListCompare
- Do not copy ListCompare.exe by itself to another folder
